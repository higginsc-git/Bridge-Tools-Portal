Bridge Moving Load Analysis Tools

Browser-based analysis tools for moving load envelopes, influence lines, and design envelopes on simply-supported spans with optional cantilevers.

Christopher C. Higgins, PhD, PE · School of Civil and Construction Engineering · Oregon State University

Each tool below is a single, self-contained HTML file. Click Launch tool to run it directly in your browser, or View on GitHub to see the README, source code, or report an issue. No installation, no account, no telemetry; all computation runs locally on your machine. Any analysis run can be saved as PDF through your browser's print dialog.

Tools

HL-93 Bridge Moving Load Analysis

Highway

AASHTO LRFD HL-93 design vehicular live load. Design tandem, design truck (with the rear-axle spacing swept from 14 to 30 ft), and design lane load. Moment and shear envelopes with peak values and full per-section tables.

Launch tool → View on GitHub

Influence Line Analysis

General

Influence line construction and evaluation at any user-selected section, for moment and for shear (left or right of the cut). Trapezoidal-rule integration with explicit zero-crossing detection for sign-changing IL regions.

Launch tool → View on GitHub

Design Envelopes for Uniform Loads

General

Design envelope analysis for uniform live load patterns, including pattern loading considerations across the structure with the worst-case effects reported at every section.

Launch tool → View on GitHub

Train Moving Load Analysis

Railway

AREMA Ch.15 Cooper E-XX (1 to 3 locomotives, with the trailing 8 k/ft uniform load) and AREMA Alternate Live Load. Includes dead load tab and AREMA-default dynamic impact factor with user override.

Launch tool → View on GitHub

About

These tools are based on an original 2004 Fortran moving load analysis program developed by the author for teaching and research in bridge engineering. The browser versions reimplement and extend that engine in JavaScript with interactive plots, influence-line analysis, and AREMA railway loadings. Each tool lives in its own GitHub repository, linked from the View on GitHub button on each card above, where you can read the README, browse source code, or report issues.
